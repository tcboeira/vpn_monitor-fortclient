# Política de Segurança

Se você identificar uma vulnerabilidade de segurança neste projeto, por favor reporte de forma responsável.

## Reportar Vulnerabilidades

Envie um e-mail para:

tcboeira@gmail.com

Incluindo:

- Descrição da vulnerabilidade
- Passos para reproduzir
- Impacto potencial

## Escopo

Este projeto é destinado para uso educacional e pessoal, portanto não há garantias formais de segurança.

Ainda assim, relatórios responsáveis são bem-vindos.
